# Git/GitHub working directory

For this project, **run all Git and GitHub commands from**:

```powershell
cd "C:\Teachers help\teachers_help"
```

Examples:

```powershell
cd "C:\Teachers help\teachers_help"
git status
git add .
git commit -m "Your message"
git push
```

If you run `git` from `C:\Teachers help` you’ll see `fatal: not a git repository`, because the repo lives in the `teachers_help` subfolder.
